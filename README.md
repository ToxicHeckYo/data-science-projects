### Описание проекта    
Угадать загаданное компьютером число за минимальное число попыток.

:arrow_up:[к оглавлению](#оглавление-)


### Какой кейс решаем?    
Нужно написать программу, которая угадывает число за минимальное число попыток

**Условия задачи:**  
- Компьютер загадывает целое число в заданном диапазоне (по умолчанию от 1 до 100), и нам его нужно угадать. Под «угадать», подразумевается «написать программу, которая угадывает число».
- Алгоритм учитывает информацию о том, больше ли случайное число или меньше нужного нам.

**Метрика качества**     
Результаты оцениваются по среднему количеству попыток при 1000 повторений

**Что практикуем:**    
- Написание качественного кода на Python
- Работа с окружением и зависимости Python
- Работа с Jupyter Notebook
- Работа с git

### Краткая информация о данных
Для проверки работы алгоритма используется список целых чисел (1000 чисел) генерируемый генератором случайных чисел с указанием диапазона значений
  
:arrow_up:[к оглавлению](#оглавление-)


### Этапы работы над проектом  
1. Создание проекта в среде разработки и его инициализация в Git
2. Создание функции для угадывания/поиска числа с подсчетом количества попыток
3. Создание функции для сбора среднего количества попыток для угадывания/поиска числа
4. Создание ноутбука для работы с функцией среднего количества попыток
5. Выгрузка зависимостей в файл
6. Создание Readme
7. Загрузка проекта на GitHub
